## CLI Usage

```
$ jira {search}
$ jira MJP-1234

$ jira issue {search}
$ jira issue #1234

$ jira board {search}
$ jira board {name} issue {search}
$ jira board {name} issue #1234

$ jira project {project} issue {search}
$ jira project {project} issue #1234
```
